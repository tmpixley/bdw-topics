Source Control Management

A source control management system (SCM) is software that provides coordination and services between members of a software development team,provides file management and version control so that team members don’t write over each other’s changes, and gives developers the ability to work concurrently on files.

In the article Source Code Management Systems: Trends, Analysis and Best Features by Janel Garvin, different SCMs are discussed, and the pros and cons of each explored. Github is not mentioned as this is an older article, but this gives an interesting history and comparison of source control management systems.
